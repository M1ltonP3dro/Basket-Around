# Basket-Around
*Conceito do Projecto:
Uma plataforma web para localizar quadras desportivas de basketball, visualizar informações
detalhadas, verificar horários disponíveis e agendar jogos. Donos de quadras podem registar seus
espaços, configurar horários e gerir reservas. O sistema também sugere quadras próximas com
base na localização do usuário.

*Ideia Geral de Funcionamento
O usuário (padrão) após cadastrar-se, terá acesso à quadras desportivas com base na sua localização, ver suas informações como horário, fotos, avaliações e endereço, agendar jogos com base no horário disponível, cancelar reservas e avaliar a quadra após a sua utilização.

O proprietário local ( admin local) irá cadastrar a sua quadra,fotos e localização, também será responsável por agendar os horários disponíveis e confirmar ou recusar reservas.

Já administrador geral será visualizar as quadras pendentes e vai aprovar ou recursar os seus registros e caso não estejam em condições para praticar atividades desportivas ou caso ela seja falsa ele poderá removê-la.

*Motivo do Projecto: 
Este projecto surgiu da dificuldade em se localizar locais apropriados para poder jogar basketball, sendo esta uma das modalidades mais praticadas em Angola. Este projecto poderá tornar-se numa rede para todos os praticantes desta modalidade.
